# SDS09 (CURA, Kuma + Laurence pop-up) - Locked Prompts

Filled from `PROMPT-LIBRARY.md`, anchored to **Kengo Kuma** (the how: layered natural
materials, timber and bamboo, tactile craft) and **Janet Laurence** (the why: biophilia,
living botanicals, nature as care). Set inside a Carriageworks heritage bay. Replaces the
retired PRISM / Eliasson prompt set. Lock the still first, then animate it with the still as
the start frame. Do not re-roll without a reason. Clips 5s to 10s, one camera move each.

Settings: Nano Banana Pro (Higgsfield) for stills, Kling for video, 16:9, 4K. If out of
Higgsfield credits, the same prompts run on Google Gemini.

**One Style Lock, a per-shot Setting line.** The earlier draft put "pure white seamless
#FFFFFF" in the Style Lock, but the in-bay scenes need the brick backdrop. So the Style Lock
holds everything constant (light, materials, palette, mood, quality) and each shot states its
own Setting: white seamless for the study objects, ochre-wash paper for the section, the raw
brick bay for the in-context scenes. That keeps the set cohesive without forcing a white void
behind a scene that should read in context.

**Negatives:** keep "no warping / no melting / no distortion" on the **Kling clips only**.
On the stills it is dropped (a static frame does not warp, and the term can flatten texture).
